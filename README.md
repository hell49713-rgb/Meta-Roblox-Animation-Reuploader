Roblox Auto Animation Spoofer

A simple, fast, and fully open-source tool for reuploading Roblox assets without the usual hassle.

This project started as a small personal utility and turned into something actually useful, so I decided to clean it up and release it publicly.

--------------------------------------------------

What it does

This tool automates the process of reuploading Roblox assets so you don’t have to manually go through the same repetitive steps over and over again.

Currently supported:

- Animation reuploading (standard animations)
- Animation reuploading from scripts
- Sound reuploading

It handles the annoying parts for you and keeps things consistent.

--------------------------------------------------

Why this exists

Honestly, I got tired of seeing people sell “spoofers” that are basically just a couple lines of code wrapped in a UI and put behind a paywall.

There’s nothing wrong with making tools, but charging for something this simple never sat right with me.

So instead of complaining about it, me (almos) and Xix decided to just make one ourselves — free, open source, and actually usable.

No locked features, no paywalls, no nonsense.

--------------------------------------------------

Features

1. Standard Animation Reuploading
Takes existing animation IDs and reuploads them under your account automatically.

- Clean and straightforward
- Works in bulk
- Keeps things organized

2. Script Animation Reuploading
Parses animations directly from scripts and reuploads them without you needing to extract IDs manually.

- Saves time when dealing with large scripts
- Useful for developers working with prebuilt systems
- Handles multiple animations in one pass

3. Sound Reuploading
Reuploads sound assets the same way as animations.

- Simple input/output
- Works alongside animation features
- No extra setup required

Usage

1. Open up START_META.bat
   - This will automatically help you install Node.js and all required dependencies

2. Enter your Roblox cookie when prompted

3. Open your Roblox Studio game

4. Open the Command Bar and Output
   - You can find both inside the Scripts tab in Roblox Studio

5. Choose your mode:
   - Normal spoof
   - Script spoof
   - Audio spoof

6. Click "RUN META"
   - The tool will automatically start reuploading animations/sounds
